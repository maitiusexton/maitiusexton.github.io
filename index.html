Identifying Handwritten Arabic Characters Using Convolutional Neural Networks

For my Data Science Immersive Capstone, I analyzed 17,000 images of handwritten Arabic characters to create a multiclass classification model with an accuracy rating of 94.17% on unseen data. OCR (Optical Character Recognition) technology has countless applications to problems in a variety of industries. These include data entry, information extraction translation, and more. Most of these technologies are based on languages which use Latin scripts. While it is true that the Latin script is the most widely used writing system in the world, this still leaves out millions of people who primarily use other scripts. Arabic is the 5th most spoken language in the world and several other languages (Farsi, Kurdish, Urdu, Uyghur, Pashto, and many more) use an Arabic-based alphabet. These reasons create a clear motivation for the creation of OCR software that can read Arabic.

Natural Language Processing on Reddit

For this project, I wanted to utilize natural language processing and various machine learning models to determine how well I can delineate different texts from similar sources. To test this, I used the PushShift API to pull over 300,000 posts from two subreddits (forums on reddit.com): r/travel and r/solotravel. These topics are obviously quite similar so I knew it would be a challenge to create an accurate model, but I was able to achieve about a 75% accuracy rating across the board.
